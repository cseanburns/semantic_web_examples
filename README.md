# Semantic Web Examples

This repository contains the companion example files for my
[ICT420: Semantic Web Development Course](https://cseanburns.github.io/semantic_web_development/).
It is a static collection of HTML and CSS demos used alongside the book and course materials.

The repository was reorganized to make the examples easier to browse and maintain:

- The main entry point is now [`index.html`](index.html) at the document root.
- Examples are grouped by lesson under [`examples/`](examples/).
- Example filenames were renamed to better describe the concept they demonstrate.
- Large catch-all pages were split into smaller, lesson-focused examples where needed.
- Legacy root files and older subdirectory paths remain as thin redirect pages so existing links still resolve.

If you want to browse the examples by topic, start at [`index.html`](index.html).
